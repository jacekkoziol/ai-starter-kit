# AGENTS.md

This repo is the **home of the AI kit** (the `AI/` folder that gets vendored into other repos).

If you are an agent (or human) working in this repo, you are **changing the kit itself** —
adding or editing rules, fields, skills, reference docs, or templates.

**Read [`MAINTAINING.md`](MAINTAINING.md) first.** It holds the prime directives, layout,
sync rules, invariants, and consistency checks for evolving the kit. Don't edit anything under
`AI/` until you've read it.

> Note: [`MAINTAINING.md`](MAINTAINING.md) is about *evolving* the kit. To learn how the kit is
> *used* on a project, see [`AI/AGENT-INSTRUCTIONS.md`](AI/AGENT-INSTRUCTIONS.md) and
> [`AI/README.md`](AI/README.md).
